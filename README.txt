
# Whah Consulting Website (Static)

## Upload to GitHub Pages
1. Upload **all files inside this folder** to the root of your repo.
2. Enable GitHub Pages: Settings → Pages → Deploy from a branch → `main` / `/root`.
3. Visit your site at `https://<username>.github.io/<repo>/`.

If using a custom domain, set a CNAME for `www` to `<username>.github.io` and A records for the apex to GitHub IPs.
